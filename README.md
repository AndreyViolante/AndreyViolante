### 👩‍💻 About me

- 🎓 Studying Software Engineering, teaching Robotics on the side
- 🧩 I like taking things apart — literally: my main side project is a reverse-engineered protocol for an educational robot with no public docs
- ⚙️ Backend-leaning, comfortable across the stack: Python/FastAPI services, React/TypeScript frontends
- 📡 Into low-level stuff too: HID protocols, byte-level frame parsing, USB capture analysis
- 🌱 Currently exploring message brokers (RabbitMQ, Kafka) and containerized service architectures

**[est-robot](https://github.com/AndreyViolante/est-robot)**

Reverse-engineered the HID protocol of an educational robot (Dr. Luck EST) by sniffing USB traffic, then wrote a Python compiler that turns plain Python scripts into the robot's byte-level program format — no proprietary IDE required.

`python` `reverse-engineering` `robotics` `hid-protocol`

<td width="50%">

**[estudos-backend](https://github.com/AndreyViolante/estudos-backend)**

Three REST APIs built with FastAPI, covering different backend patterns: relational CRUD (PostgreSQL), integration testing against a real database, and event-driven order processing (MongoDB + RabbitMQ + Kafka), all containerized with Docker.

`fastapi` `postgresql` `mongodb` `docker` `pytest`
