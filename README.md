## About

Software Engineering student with a backend focus and a hands-on approach to learning: I build real systems to understand how they work — from REST APIs and event-driven services to mobile apps and reverse-engineered hardware protocols.

- **Backend**: Python, FastAPI, PostgreSQL, MongoDB, RabbitMQ, Kafka, Docker
- **Mobile & Frontend**: React Native (Expo), React, TypeScript
- **Low-level**: HID protocols, byte-level frame parsing, USB traffic analysis
- **Teaching**: Robotics instructor, working with Arduino and educational robotics kits

Currently deepening my knowledge of message brokers and containerized service architectures.

## Featured Projects

### [est-robot](https://github.com/AndreyViolante/est-robot)

Reverse engineering of an educational robot (Dr. Luck EST) with no public documentation. Captured and analyzed USB traffic to map the robot's HID protocol, then built a Python compiler that translates plain Python scripts into the robot's byte-level program format — removing the dependency on the proprietary IDE.

`Python` · `Reverse Engineering` · `HID Protocol` · `Robotics`

### [EnglishTutor](https://github.com/AndreyViolante/EnglishTutor)

AI-powered English tutoring app for Brazilian middle school students (grades 6–8). An animated virtual teacher holds conversations by text and voice — speech recognition for the student, text-to-speech for the teacher — adapting content to the student's grade and curriculum, with explanation and role-play modes. Built with React Native, Expo and LLaMA 3.3 via the Groq API.

`React Native` · `Expo` · `TypeScript` · `LLM` · `Speech Recognition`

### [estudos-backend](https://github.com/AndreyViolante/estudos-backend)

Three REST APIs exploring distinct backend patterns: relational CRUD with PostgreSQL, integration testing against a real database with pytest, and event-driven order processing with MongoDB, RabbitMQ and Kafka. All services containerized with Docker.

`FastAPI` · `PostgreSQL` · `MongoDB` · `RabbitMQ` · `Kafka` · `Docker` · `pytest`
